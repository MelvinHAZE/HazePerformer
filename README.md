# 🎛️ HazePerformer

**HazePerformer** est une suite de mappings MIDI open source pour **Mixxx**.

Son objectif est de transformer des contrôleurs MIDI abordables en véritables surfaces de performance pour DJ, live remix et finger drumming.

---

## 🎚️ Contrôleurs supportés

- ✅ Akai MPK Mini MK3
- 🚧 Akai APC Mini
- 🚧 Akai LPD8

---

## 🚀 Fonctionnalités prévues

- Smart Jog
- Hot Cues
- Loops
- Samplers
- Effets
- Navigation bibliothèque
- Mode Performance
- Mode Remix
